# 🌍 Global Country Finder

This is a simple React.js and Tailwind CSS web application that connects to a live Public API to fetch and display information about all countries around the world.

### 🌟 What this app does:
* **Live Data:** It fetches real-time data like country flags, capital cities, regions, and population from the free REST Countries API.
* **Smart Search:** You can type any country name in the search bar, and it filters the results instantly.
* **Loading Spinner:** It shows a clean "Loading..." animation while fetching the data so the screen never looks blank.
* **Error Handling:** If there is no internet connection, it shows a friendly error message with a "Try Again" button.

### 🛠️ Tech Used:
* React.js (useState & useEffect)
* Tailwind CSS
* REST Countries API
